# 📊 Sales & Revenue Analysis Dashboard

## 📌 Overview
This project is an interactive Sales & Revenue Analysis Dashboard built using Microsoft Power BI using the Sample Superstore dataset.

## 🚀 Features
- 📈 Total Sales KPI
- 💰 Total Profit KPI
- 📦 Total Quantity KPI
- 🎯 Total Discount KPI
- 📊 Sales Trend Analysis
- 🛍️ Sales by Category
- 🌍 Sales by Region
- 🔝 Top 10 Products by Sales
- 🎛️ Interactive Slicers (Category, Region, Order Date)

## 🛠 Tools Used
- Microsoft Power BI
- Sample Superstore Dataset
